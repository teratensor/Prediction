# 3_insights

당첨번호 패턴 분석 및 확률 분포 생성

## 구조

각 인사이트는 `{name}/` 폴더로 구성:
- `generate.py`: 인사이트 생성 로직
- `config.json`: 메타데이터 및 요약 통계
- `distribution.json`: 확률 분포 (key: 값, value: 확률)

## 현재 인사이트

- **sum/**: 당첨번호 6개 합계 분포

## 철학

확률 기반 생성 (필터링 아님)
- 모든 8,145,060개 조합이 생성 가능
- 역대 패턴 → 높은 확률
- 이례적 패턴 → 낮은 확률 (0은 아님)
