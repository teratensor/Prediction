# Consecutive(연속수) 인사이트

당첨번호 내 연속수 패턴 분석

## 실행

```bash
python generate.py
```

## 분석 항목

1. 연속쌍 개수 분포 (0쌍, 1쌍, 2쌍, ...)
2. 연속수 시퀀스 개수 분포
3. 연속수 길이별 분포 (2연속, 3연속, ...)
4. 연속수 시작 번호 분포
5. 연속수 시작 위치 분포
6. 연속수 패턴 상세
7. 최근 트렌드 분석

## 출력 파일 (statistics/)

| 파일 | 설명 |
|------|------|
| consecutive_pair_distribution.csv | 연속쌍 개수별 분포 |
| sequence_count_distribution.csv | 시퀀스 개수별 분포 |
| consecutive_length_distribution.csv | 연속수 길이별 분포 |
| consecutive_start_value_distribution.csv | 시작 번호 분포 |
| consecutive_start_position_distribution.csv | 시작 위치 분포 |
| consecutive_pattern_distribution.csv | 패턴 상세 |
| consecutive_trend_analysis.csv | 최근 트렌드 |
| summary.csv | 요약 통계 |

## 연속수 정의

- **2연속**: (n, n+1) 예: 5, 6
- **3연속**: (n, n+1, n+2) 예: 10, 11, 12
- **다중 연속**: 2쌍 이상 예: 5,6과 20,21

## 주요 통계

- **연속수 있는 회차**: 52.2%
- **연속수 없는 회차**: 47.8%
- **회차당 평균 연속쌍**: 0.65개

## 핵심 발견

### 1. 연속수 시작 위치 (균등 분포)
- 위치 5 (ball5-ball6): 22.5%
- 위치 4 (ball4-ball5): 22.0%
- 위치 1 (ball1-ball2): 20.7%

### 2. 연속수 시작 번호 상위
- 38, 17, 30: 각 11개 (4.8%)
- 35: 9개 (4.0%)
- 6, 3: 각 8개 (3.5%)

### 3. 연속수 패턴
- 없음: 47.8%
- 2연속 1개: 40.9%
- 2연속 2개: 6.6%
- 3연속 1개: 3.4%
