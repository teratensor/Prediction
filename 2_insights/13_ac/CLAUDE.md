# AC (Arithmetic Complexity) 인사이트

6개 당첨번호의 산술적 복잡성 분석

## AC값 계산 방법

```
AC = (모든 쌍의 차이값 중 유니크 개수) - 5
```

- 6개 번호 → 15개 쌍 (6C2)
- 최소 AC: 0 (연속 6개, 예: 1,2,3,4,5,6)
- 최대 AC: 10 (15개 차이값 모두 다름)

### 계산 예시

번호: (1, 2, 12, 23, 40, 41)

차이값 (15개):
- 41-40=1, 41-23=18, 41-12=29, 41-2=39, 41-1=40
- 40-23=17, 40-12=28, 40-2=38, 40-1=39
- 23-12=11, 23-2=21, 23-1=22
- 12-2=10, 12-1=11
- 2-1=1

유니크: {1, 10, 11, 17, 18, 21, 22, 28, 29, 38, 39, 40} = **12개**

**AC = 12 - 5 = 7**

### 필터 규칙

```python
# 조합 생성 시 AC값 체크
def is_valid_ac(balls, min_ac=6):
    differences = set()
    for i in range(6):
        for j in range(i+1, 6):
            differences.add(abs(balls[j] - balls[i]))
    ac = len(differences) - 5
    return ac >= min_ac
```
