# One-Hot Encoding 인사이트

ord1~ord6를 45비트 원핫인코딩으로 변환하여 패턴 분석

## 실행

```bash
python generate.py
```

## 분석 항목

1. ord별 45비트 원핫인코딩 생성
2. 포지션별(ord1~6) 비트 활성화 빈도
3. 비트 쌍 동시 활성화 분석
4. 클러스터별 활성화 패턴 (C1~C5)
5. 희소성(Sparsity) 분석
6. 비트 시퀀스 패턴 (연속 비트)
7. 포지션 간 비트 차이 분석
8. 핫/콜드 비트 분석
9. 최근 트렌드 분석
10. ML용 피처 매트릭스 생성

## 출력 파일 (statistics/)

| 파일 | 설명 |
|------|------|
| onehot_matrix.csv | 전체 원핫인코딩 매트릭스 |
| bit_frequency_by_position.csv | 포지션별 비트 활성화 빈도 |
| bit_pair_frequency.csv | 비트 쌍 동시 활성화 빈도 |
| cluster_pattern_distribution.csv | 클러스터 패턴 분포 |
| cluster_by_position.csv | 포지션별 클러스터 분포 |
| sparsity_analysis.csv | 희소성 분석 |
| bit_sequence_pattern.csv | 연속 비트 패턴 |
| bit_gap_distribution.csv | 비트 간격 분포 |
| position_bit_difference.csv | 포지션 간 비트 차이 |
| hot_cold_bits.csv | 핫/콜드 비트 목록 |
| bit_trend_analysis.csv | 비트 트렌드 |
| ml_feature_matrix.csv | ML용 270 피처 매트릭스 |
| summary.csv | 요약 통계 |

## 원핫인코딩 구조

- **포지션당 비트**: 45
- **총 비트**: 270 (6 포지션 × 45 비트)
- **활성 비트**: 6개/회차
- **희소성**: 97.78%
