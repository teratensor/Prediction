# 합계(Sum) 인사이트

당첨번호 6개의 합계 분석

## 실행

```bash
python generate.py
```

## 분석 항목

1. 합계 기본 빈도 분석
2. 합계 구간별 분포 (5구간)
3. 10단위 구간 분포
4. 합계 끝자리 분석
5. 이론 vs 실제 분포 비교
6. 합계별 평균 범위(span) 분석
7. 최근 트렌드 분석

## 출력 파일 (statistics/)

| 파일 | 설명 |
|------|------|
| sum_distribution.csv | 합계별 분포 |
| sum_range_distribution.csv | 합계 구간별 분포 |
| sum_10unit_distribution.csv | 10단위 구간 분포 |
| sum_lastdigit_distribution.csv | 합계 끝자리 분포 |
| sum_theory_vs_actual.csv | 이론 vs 실제 비교 |
| sum_span_relation.csv | 합계별 범위 관계 |
| sum_trend_analysis.csv | 최근 트렌드 |
| summary.csv | 요약 통계 |
