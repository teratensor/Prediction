# Consecutive(연속수) 인사이트

당첨번호 내 연속수 패턴 분석

## 실행

```bash
python generate.py
```

## 분석 항목 (예정)

1. 연속수 개수 분포 (0개, 2개, 3개, ...)
2. 연속수 시작 위치 분석
3. 연속수 시작 번호 분포
4. 다중 연속수 패턴 (2쌍 이상)
5. 최근 트렌드 분석

## 연속수 정의

- 2개 연속: (n, n+1) 예: 5,6
- 3개 연속: (n, n+1, n+2) 예: 10,11,12
- 다중 연속: 2쌍 이상 예: 5,6과 20,21

## 참고

- 약 50% 회차에서 연속수 출현
- 2개 연속이 가장 빈번
