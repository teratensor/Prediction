# 구간(Range) 인사이트

당첨번호의 구간별 분포 분석

## 구간 정의

| 구간 | 범위 | 크기 |
|------|------|------|
| 1-10 | 1~10 | 10개 |
| 11-20 | 11~20 | 10개 |
| 21-30 | 21~30 | 10개 |
| 31-40 | 31~40 | 10개 |
| 41-45 | 41~45 | 5개 |

## 실행

```bash
python generate.py
```

## 출력 파일 (statistics/)

| 파일 | 설명 |
|------|------|
| pattern_distribution.csv | 구간 패턴(5자리 코드) 분포 |
| range_count_distribution.csv | 각 구간별 개수 분포 |
| summary.csv | 요약 통계 (구간별 평균) |

## 패턴 코드 형식

5자리 숫자 (예: `21210`)
- 각 자리: 해당 구간에 포함된 번호 개수
- 순서: 1-10, 11-20, 21-30, 31-40, 41-45
- 예: `21210` = 1-10구간 2개, 11-20구간 1개, 21-30구간 2개, 31-40구간 1개, 41-45구간 0개
