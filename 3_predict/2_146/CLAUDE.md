# 146 인사이트 (ord1, ord4, ord6)

첫수(ord1), 넷째수(ord4), 끝수(ord6) 조합 분석

## 실행

```bash
# 1. firstend에서 result.csv 생성 (ord1, ord6 쌍)
python ../1_firstend/export_result.py

# 2. ord4 채우기 (공식별 행 복제)
python fill_ord4.py

# 3. 통계 분석 (선택)
python generate.py
```

## 입출력

**입력**: `result/result.csv` (ord1, ord6 쌍 - 1_firstend에서 생성)

**출력**: `result/result.csv` (ord4 채움)

| 컬럼 | 설명 |
|------|------|
| ord1 | 첫수 (1-21) |
| ord2 | (null) |
| ord3 | (null) |
| ord4 | 넷째수 (공식으로 계산) |
| ord5 | (null) |
| ord6 | 끝수 (23-45) |
| 빈도수 | 출현 횟수 |
| 회차 | 출현 회차 목록 |
| 공식 | A, B, C 구분 |

## ord4 예측 공식

| 공식 | 비율 | 설명 | 적중률 |
|------|------|------|--------|
| A | 0.60 | 기본 (ord1~ord6 범위의 60% 지점) | 74.7% |
| B | 0.31 | 낮음 (ord4가 ord1에 가까움) | 15.3% |
| C | 0.87 | 높음 (ord4가 ord6에 가까움) | 12.9% |

**계산식**: `ord4 = ord1 + (ord6 - ord1) × 비율`

## 처리 규칙

1. 각 (ord1, ord6) 쌍에 대해 3개 공식(A, B, C) 적용
2. ord4 유효성: ord1 < ord4 < ord6
3. **(ord1, ord4, ord6) 중복 금지**
4. 빈도순 정렬

## 통계

- 입력: 477개 행 (ord1, ord6 쌍)
- 출력: 1,431개 행 (공식별 복제)
- 공식별: A 477개, B 477개, C 477개
