# 146 - ord4 계산

첫수(ord1), 넷째수(ord4), 끝수(ord6) 조합 생성

**참고**: 현재 main.py에서 통합 처리됨. 이 폴더의 fill_ord4.py는 레거시 버전.

```
main.py (통합)
```

## 현재 설정 (main.py v1.01)

| 파라미터 | 값 | 설명 |
|----------|-----|------|
| FORMULA_RATIO | 0.60 | 공식 A 비율 |
| OFFSET_RANGE | ±10 | offset 범위 (-10 ~ +10) |
| 출력 | ~9,000개 | (ord1, ord4, ord6) 행 |

## ord4 계산 공식

```python
base_ord4 = round(ord1 + (ord6 - ord1) × 0.60)
ord4 = base_ord4 + offset  # offset: -10 ~ +10
```

### 유효성 검사

```python
ord1 < ord4 < ord6
```

## 이전 방식 (레거시)

| 공식 | 비율 | 설명 |
|------|------|------|
| A | 0.60 | 기본 |
| B | 0.31 | 낮음 |
| C | 0.87 | 높음 |

→ 현재는 공식 A + ±10 offset으로 대체됨
