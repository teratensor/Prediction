# 2_predict

가중치 기반 점수 시스템으로 로또 번호 예측

## 실행

```bash
# 다음 회차 예측
python predict.py

# 특정 회차 예측
python predict.py --round 1204
```

## 출력

- **Top24**: 고확률 번호 24개
- **Mid14**: 중간 확률 번호 14개
- **Rest7**: 저확률 번호 7개

## 점수 요소

| 요소 | 설명 | 점수 |
|------|------|------|
| FR (빈도) | 출현 빈도 기반 | 0~4500 |
| CR (이월) | 직전 1~4회차 출현 | 12000~100000 |
| MOD (끝수) | 상위 빈도 끝수 | 80~150 |
| PR (소수) | 소수 여부 | 100 |
| RG (범위) | 1-15/16-30/31-45 밸런스 | 27~400 |

## 핵심 함수

- `load_data()`: winning_numbers.json 로드
- `predict()`: 점수 계산 → Top24/Mid14/Rest7 분류
